# Export Architecture as a Sankey Diagram

Visualize, analyze, and export the complexity of a hierarchical system architecture (Simulink models of different systems, subsystems, components, and their variants, etc.) using a Sankey diagram.

## Example

A MATLAB live script, [`sankey.mlx`](sankey.mlx), within this directory serves as an example to visualize the complexity of `{'Perception', 'Planning', 'Control', 'Simulation'}` subsystem models of the `AutoDRIVE_RZR_AEB` system architecture using a Sankey diagram.
